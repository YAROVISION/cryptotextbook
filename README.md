# Cryptotextbook (Підручник з Крипто-скальпінгу)

Репозиторій містить структурований освітній підручник-довідник з **крипто-скальпінгу та високочастотної торгівлі (HFT)** на мікро-таймфреймах. Усі матеріали систематизовано для вивчення механіки ринку, торгових стратегій, управління ризиками та психологічної дисципліни.

Матеріали адаптовані для використання в Obsidian (з підтримкою внутрішніх посилань вікі-формату `[[Розділ]]`).

---

## 📚 Структура підручника

Весь контент знаходиться в папці `book/` та поділений на наступні розділи:

1. **[Розділ 1. Фундамент скальпера та механіка ринку](book/%D0%A0%D0%BE%D0%B7%D0%B4%D1%96%D0%BB%201.%20%D0%A4%D1%83%D0%BD%D0%B4%D0%B0%D0%BC%D0%B5%D0%BD%D1%82%20%D1%81%D0%BA%D0%B0%D0%BB%D1%8c%D0%BF%D0%B5%D1%80%D0%B0%20%D1%82%D0%B0%20%D0%BC%D0%B5%D1%85%D0%B0%D0%BD%D1%96%D0%BA%D0%B0%20%D1%80%D0%B8%D0%BD%D0%BA%D1%83.md)**
   * Роль скальпера на ринку, базова формула професійного скальпінгу (паливо ринку, робота з неефективностями, аналіз потоку ордерів) та анатомія біржового стакана (Ask/Bid, лімітні та комерційні ордери).
   * Робота зі стрічкою угод та кластерами (Footprint, POC).
   * Концепція доставки ціни: імбаланси (FVG) та пули ліквідності.

2. **[Розділ 2. 3. **[Розділ 3. Ризик-менеджмент у високочастотній торгівлі](book/%D0%A0%D0%BE%D0%B7%D0%B4%D1%96%D0%BB%203.%20%D0%A0%D0%B8%D0%B7%D0%B8%D0%BA-%D0%BC%D0%B5%D0%BD%D0%B5%D0%B4%D0%B6%D0%BC%D0%B5%D0%BD%D1%82%20%D1%83%20%D0%B2%D0%B8%D1%81%D0%BE%D0%BA%D0%BE%D1%87%D0%B0%D1%81%D1%82%D0%BE%D1%82%D0%BD%D1%96%D0%B9%20%D1%82%D0%BE%D1%80%D0%B3%D1%96%D0%B2%D0%BB%D1%96.md)**

4. **[Розділ 4. Специфіка крипторинку](book/%D0%A0%D0%BE%D0%B7%D0%B4%D1%96%D0%BB%204.%20%D0%A1%D0%BF%D0%B5%D1%86%D0%B8%D1%84%D1%96%D0%BA%D0%B0%20%D0%BA%D1%80%D0%B8%D0%BF%D1%82%D0%BE%D1%80%D0%B8%D0%BD%D0%BA%D1%83.md)**

## 💻 Локальний графічний інтерфейс (Reader & Editor)

У репозиторій інтегровано локальний вебдодаток для зручного читання та редагування підручника з преміальним дизайном (тепла тема пергаменту за референсом Anthropic, моноширинні шрифти JetBrains Mono та гласморфізм).

### Функціонал додатка

* 📖 **Режим читання (Read Mode)**: красивий рендеринг Markdown, LaTeX-формул, підсвічування коду та GitHub-попереджень (`[!NOTE]`, `[!TIP]` тощо) з інтерактивним бічним змістом.
* ✍️ **Режим редагування (Edit Mode)**: вбудований редактор **Monaco Editor** (ядро VS Code) з підсвічуванням синтаксису.
* 🖥️ **Розділений екран (Split view)**: зліва редактор, справа — оновлення рендерингу в реальному часі зі синхронізованим прокручуванням.
* 🔍 **Глобальний пошук**: миттєвий офлайн-пошук ключових слів за всіма розділами книги з підсвічуванням збігів.
* 💾 **Автозбереження та бекапи**: автоматично зберігає ваші зміни та створює резервну копію попереднього файлу в папці `reader/backups/` перед перезаписом.
* 📱 **Адаптивність**: повністю адаптований інтерфейс із висувним меню-шторкою для зручного читання на смартфонах.

### Як запустити

1. Перейдіть у папку додатка:

   ```bash
   cd reader
   ```

2. Інсталюйте залежності (потрібен Node.js):

   ```bash
   npm install
   ```

3. Запустіть сервер:

   ```bash
   npm run dev
   ```

4. Відкрийте у браузері: **<http://localhost:3000>**

## 🤖 Автоматичний збір новин (last30days-skill)

У проект інтегровано навичку `last30days-skill` для автоматичного збору ринкових подій, трендів та обговорень за останні 30 днів (з Reddit, Hacker News, Polymarket, GitHub та Web). Зібрані новини використовуються для наповнення [Розділу 7](book/%D0%A0%D0%BE%D0%B7%D0%B4%D1%96%D0%BB%207.%20%D0%A9%D0%BE%D0%B4%D0%B5%D0%BD%D0%BD%D0%B8%D0%B9%20%D0%BE%D0%B3%D0%BB%D1%8F%D0%B4%20%D0%BF%D0%BE%D0%B4%D1%96%D0%B9.md).

### Як користуватись

1. **Через діалог з AI-помічником (Рекомендовано)**
   Просто попросіть вашого AI-помічника виконати пошук у чаті. Наприклад:
   * *«Збери новини за останні 30 днів про Ethereum ETF за допомогою скіла»*
   * *«Використай скіл last30days для аналізу сентименту навколо монети HYPE»*
   Помічник самостійно побудує план запиту, запустить дослідження та згенерує структуровану картку новин за шаблоном Розділу 7.

2. **Самостійно через термінал (CLI)**
   Ви можете викликати скрипт безпосередньо у вашому терміналі з папки проекту (потребує Python 3.12+):
   * **Простий збір даних по темі:**

     ```bash
     python3.12 .agents/skills/last30days/scripts/last30days.py "ваша тема"
     ```

   * **Пошук трендів у певній сфері (Discovery mode):**

     ```bash
     python3.12 .agents/skills/last30days/scripts/last30days.py --discover "cryptocurrency"
     ```

   * **Перегляд довідки та прапорців:**

     ```bash
     python3.12 .agents/skills/last30days/scripts/last30days.py --help
     ```

*Детальні налаштування збереження звітів (memory dir) та інтеграції X/Twitter чи ScrapeCreators описані в оригінальній інструкції навички [.agents/skills/last30days/SKILL.md](.agents/skills/last30days/SKILL.md).*

### 📋 Автоматичний моніторинг списку акаунтів (Watchlist)

У проект додано можливість відстежувати фіксований перелік акаунтів та джерел:

* **Конфігурація:** Список джерел зберігається у файлі [.agents/watchlist.json](.agents/watchlist.json). Ви можете вільно редагувати його, додаючи X-акаунти, GitHub-профілі та сабредіти.
* **Запуск через AI-помічника:** Просто попросіть мене: *«Перевір мій вотчліст»*.
* **Запуск через термінал (CLI):**

    ```bash
    python3.12 scripts/check_watchlist.py
    ```

    *(Для офлайн-тестування використовуйте прапорець `--mock`)*.
